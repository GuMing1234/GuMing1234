# 小说下载
## 介绍：
* 本栏目将会保存一些常见的免费小说的爬取，当然了，有些付费的小说网站因为本人技术栈有限，无法爬取，求谅解。

## 使用注意事项：
### 一 . 笔趣阁下载
1. 笔趣阁的小说比较简单就能爬取，没有什么复杂的地方，使用基础的beautifulsoup就能很简单的爬取。
2. 但笔趣阁小说有个特点，网站的地址很容易变，导致无法爬取，我尽量做到定时检查修改，如果发现无法爬取，记得检查网址和我源代码里面的网址是否一致，可能会出现我源代码里的网址输到浏览器但requests无法得到html页面，这通常是出现了重定向现象(服务器针对某个网址给我们重新定位个新的网址)
3. 注意事项用pip下载相应的包。
